# True ranking of the weights of 20 potatoes.

True ranking of the weights of 20 potatoes.

## Usage

``` r
potato_true_ranking
```

## Format

An object of class `numeric` of length 20.

Learning from Preference Data.” *Annual Review of Statistics and Its
