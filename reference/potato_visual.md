# Potato weights assessed visually

Result of ranking potatoes by weight, where the assessors were only
allowed to inspected the potatoes visually. 12 assessors ranked 20
potatoes.

## Usage

``` r
potato_visual
```

## Format

An object of class `matrix` (inherits from `array`) with 12 rows and 20
columns.

## References

Liu Q, Crispino M, Scheel I, Vitelli V, Frigessi A (2019). “Model-Based
Learning from Preference Data.” *Annual Review of Statistics and Its
Application*, **6**(1).
[doi:10.1146/annurev-statistics-031017-100213](https://doi.org/10.1146/annurev-statistics-031017-100213)
.
